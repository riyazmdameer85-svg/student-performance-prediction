# 🎓 Student Performance Prediction

## 📌 Overview
This project predicts whether a student will pass or fail based on factors like study hours, attendance, sleep hours, previous marks, and practice score using Machine Learning.

---

## 🚀 Features
- Predict student performance (Pass/Fail)
- Uses multiple input features
- Model evaluation using classification metrics

---

## 🛠️ Tech Stack
- Python
- pandas
- scikit-learn
- Flask (if you used web app)

---

## 📊 Model Used
- Decision Tree Classifier

---

## 📈 Workflow
1. Collect input data
2. Preprocess features
3. Train classification model
4. Make prediction
5. Evaluate using Precision, Recall, F1 Score
